# HoloOffice

Prototype management game inspired by running a Hololive-style entertainment company in a top-down 2D view.

## Current foundation

- Grid-based top-down headquarters with walls, doors, floors, and room designation.
- Designation-ready support spaces: office, bathroom, and storage.
- Fast in-game clock with pause and speed controls. One day lasts about five real minutes at normal speed.
- Workers with monthly salaries, mascot sprites, and simple roaming behavior.
- Talents with unlock levels, contracts, assignment tasks, and talent sprites.
- Centralized localization keys prepared for English, Spanish, and Japanese.

## Controls

- Left click inside the map to build or designate.
- Right click to cancel the current build mode.
- Left click a worker or talent sprite to select them.
- Use the right-side panel to recruit staff, assign talent work, and switch language.

## Notes

This is a gameplay foundation intended to be expanded with pathfinding, enclosed-room validation, production chains, UI polish, save/load, and deeper company progression.
